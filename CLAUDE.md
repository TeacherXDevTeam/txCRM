Internal CRM platform for TeacherX operations — school partnerships, training management, sales pipeline, meeting notes, and trainer tracking. Built for the internal team (ops, sales, content, training).

## Stack

- **Frontend:** Next.js 14 (App Router, TypeScript strict)
- **UI:** Tailwind CSS + shadcn/ui (`components/ui/`)
- **Backend:** Next.js API Routes (tRPC if complexity grows)
- **Database:** Supabase (PostgreSQL + Auth + RLS + Realtime)
- **Rich text:** Tiptap (M5 meeting notes)
- **Search:** Supabase FTS (`tsvector` on `meetings.notes`, `simple` config for Turkish)
- **Deployment:** Vercel

## Database

20+ tables across 10 modules (full schema in `teacherx_crm_prd.md §4`). Key tables:

```
contacts, schools, coordinators, onboarding_milestones
leads, activities
trainings, packages, package_trainings, trainers, assignments
contracts, orders
meetings, meeting_contacts, todo_items
working_groups, wg_members, wg_sessions, wg_phases
team_members
```

RLS is active on all tables. Always use the server Supabase client for mutations.

## Auth & Roles

- Supabase Auth (email/password, invite-only — admin creates users)
- Role stored in `team_members.role`: `admin | member | viewer`
- RLS policies join on `auth.uid()` → `team_members.id`
- Leads: only `assigned_to` + admin can read
- Contracts: only admin + operasyon department

## Conventions

- **Server Components by default.** Add `'use client'` only for interactivity.
- **Supabase server client:** `@supabase/ssr` package, `createServerClient(cookies())`.
- **Typed queries:** Always import from `types/database.ts` (generated). Never use `any`.
- **shadcn/ui:** Install via `npx shadcn-ui@latest add <component>`. Do not hand-edit `components/ui/`.
- **Turkish labels:** DB column names in English (snake_case), UI labels in Turkish.
- **Enum values:** Stored as Turkish slugs matching PRD (e.g. `kapandi_kazanildi`, `yuz_yuze`).
- **Date format:** ISO 8601 in DB; display format `dd.MM.yyyy` in UI.
- **Currency:** TL, displayed as `₺` with `.toLocaleString('tr-TR')`.
